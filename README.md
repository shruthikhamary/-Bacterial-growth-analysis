Bacterial Growth Analysis

 Overview
This project analyzes bacterial growth over time with and without antibiotic treatment using Python.

Objective
To compare growth patterns and understand the effect of antibiotics on bacterial populations.

Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

Methodology
- Collected time vs growth data
- Plotted growth curves
- Applied linear regression to model trends
- Compared treated vs untreated samples

Results
- Bacterial growth increases rapidly without antibiotics
- Growth rate significantly reduces with antibiotic treatment
